ğŸ” Verifikasi Integritas: Hash

Fungsi Hash mengubah input data sembarang menjadi output fixed-length (digest) yang unik dan tidak dapat dibalik.

Properti penting:

  • Deterministic â€” Input sama â†’ output selalu sama
  • One-way â€” Tidak bisa dibalik
  • Collision resistant â€” Sulit menemukan 2 input = 1 hash
  • Avalanche effect â€” 1 bit berubah â†’ hash sangat berbeda

ğŸ§ Akuisisi dengan dd (WSL/Linux)

  bs=4096 conv=noerror,sync status=progress
